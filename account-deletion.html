1개요

앱 이름: Whale Log

개발자: Whale Log

이메일: whale.log.story@gmail.com

Whale Log는 사용자의 데이터 통제권을 존중합니다. 본 페이지는 사용자가 계정 또는 개별 데이터를 직접 삭제하거나 삭제 요청을 보내는 방법을 안내합니다.

2계정 전체 삭제 방법 (회원 탈퇴)

방법 A — 앱 내에서 직접 삭제

  1. Whale Log 앱 실행 후 로그인
  2. 하단 탭 "마이" 선택
  3. 오른쪽 상단 설정(⚙) 아이콘 탭
  4. 설정 화면에서 가장 아래 "회원 탈퇴" 항목 탭
  5. 안내 확인 후 탈퇴 확정 버튼 탭

방법 B — 이메일로 요청

앱 사용이 어려운 경우 다음 이메일로 요청해 주세요.

수신: whale.log.story@gmail.com

제목: [Whale Log] 계정 삭제 요청

본문에 포함할 내용:

  • 가입 시 사용한 이메일 또는 아이디(username)
  • 본인 확인을 위한 추가 정보(가입일·자주 이용한 스팟 등)

접수 후 영업일 기준 3일 이내에 삭제 처리하고 회신해 드립니다.

3일부 데이터만 삭제하는 방법

계정 전체를 삭제하지 않고, 특정 데이터만 삭제할 수 있습니다.

다이어리 삭제

  1. 마이 → 내 다이어리 목록에서 해당 다이어리 진입
  2. 오른쪽 상단 휴지통 아이콘 탭
  3. 삭제 확인 → 즉시 제거

즐겨찾기 해제

  1. 스팟 상세 화면에서 별(★) 아이콘 탭
  2. 또는 홈 ⭐즐겨찾기 탭에서 해제

내 보드 삭제

  1. 마이 → 내 보드 섹션 진입
  2. 삭제할 보드 카드의 옵션 메뉴(⋮) → 삭제

사진만 삭제

  1. 다이어리 수정 화면에서 사진 옆 ✕ 버튼 탭
  2. 저장 → 해당 사진은 R2 스토리지에서도 즉시 자동 정리

4삭제되는 데이터 / 보관되는 데이터

구분 회원 탈퇴 시 처리 보관 기간
이메일·아이디·비밀번호 해시 즉시 익명화 없음
프로필(레벨·보드 정보 등) 즉시 삭제 없음
다이어리·메모·평점 즉시 삭제 없음
다이어리 사진(R2) 즉시 삭제 없음
즐겨찾기·내 보드·뱃지 즉시 삭제 없음
공개 다이어리 댓글 작성자 표시 "(탈퇴한 사용자)"로 익명화 댓글 본문은 다른 사용자 맥락 보존을 위해 유지
신고 처리 로그(admin_logs) 운영 기록으로 익명화 후 보관 1년
접속 로그·서버 로그 법령 의무에 따른 최소 기간 보관 후 자동 삭제 최대 3개월
⚠️ 회원 탈퇴는 되돌릴 수 없습니다. 동일 이메일로 재가입은 가능하지만 이전 데이터(다이어리·뱃지·즐겨찾기 등)는 복원되지 않습니다.

5제3자 서비스에서의 처리

다음 외부 서비스에 저장된 데이터도 함께 처리됩니다.

  • Cloudflare R2 — 다이어리 사진. 회원 탈퇴 또는 사진 삭제 시 즉시 제거.
  • Google / Kakao — 소셜 로그인으로 받은 식별자만 보관. 탈퇴 시 연결 해제 + 식별자 즉시 삭제.
  • Firebase Cloud Messaging — 푸시 알림 토큰. 탈퇴 시 즉시 삭제.

Google/Kakao 계정 자체에 저장된 정보는 각 서비스의 정책에 따라 별도로 관리되며, 해당 플랫폼의 계정 설정에서 직접 처리하실 수 있습니다.
