## 페이지

## 동작 방식

모바일 앱(`whalelog/whale-log-mobile`)의 회원가입 화면에서 `react-native-webview`로 위 두 URL을 모달 안에 띄워서 표시합니다. 약관 변경 시 본 repo의 HTML 파일만 수정하면 1~2분 내에 앱에 자동 반영됩니다.
